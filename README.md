# 🌐 Prodigy InfoTech Internship — Task 01

## Responsive Landing Page with Interactive Navigation Menu

A responsive landing page built with HTML, CSS, and JavaScript as part of the **Prodigy InfoTech Web Development Internship**.

---

## 🚀 Features

- Fixed navigation bar visible on all pages
- Navbar background changes on scroll
- Active link highlights as you scroll through sections
- Smooth hover effects on menu items
- 4 fully styled sections — Home, About, Services, Contact
- Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
